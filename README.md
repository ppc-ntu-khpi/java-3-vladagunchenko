# Практична робота "Масиви, вирази, керування виконанням програми" 3

## Опис роботи

В рамках даної практичної роботи було реалізовано алгоритм сортування
одновимірного масиву шляхом розбиття його на групи елементів розміру
1, 2, 4, 8 і т.д. з подальшим злиттям відсортованих частин.

Алгоритм реалізовано за допомогою ітеративного методу злиття (Merge Sort).

---

## Обране завдання 10

Відсортувати одновимірний масив, розбиваючи його на групи елементів по  
1, 2, 4, 8 і т.д.  
Дозволяється використовувати допоміжний масив.

---

## Реалізація

Алгоритм працює наступним чином:

1. Масив розбивається на групи розміру 1.
2. Сусідні групи зливаються у відсортовані групи розміру 2.
3. Далі формується розмір груп 4, потім 8 і т.д.
4. Процес повторюється до повного сортування масиву.

У реалізації використано:

- ітеративний підхід (без рекурсії);
- один допоміжний масив для оптимізації використання пам’яті;
- тип даних `int[]` як оптимальний для зберігання цілих чисел.

---

## Висновок

У ході виконання практичної роботи було реалізовано ефективний алгоритм
сортування масиву методом злиття з використанням ітеративного підходу.
Отримано практичні навички роботи з масивами, циклами та керуючими
конструкціями мови Java.
